# Tkinter Calculator 

A simple desktop calculator built using Python and Tkinter that allows users to perform basic arithmetic operations through an interactive graphical user interface.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Modulus
* Positive/Negative Number Toggle (+/-)
* Clear Input Functionality
* Simple and Easy-to-Use GUI

## Technologies Used

* Python 3
* Tkinter

## How to Run

1. Make sure Python 3 is installed.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:

```bash
python tkinter_calculator.py
```

## Sample Usage

```text
Enter first number
↓
Select operation (+, -, *, /, %)
↓
Enter second number
↓
Press =
↓
Display result
```

## Calculator Functions

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Modulus (%)
* Sign Toggle (+/-)
* Clear Input
* Exit Application
